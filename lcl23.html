User Guidelines

Alur Kerja LCL Selain BC 20
Penjelasan Flowchart LCL OUT SELAIN BC 2.0
  1. Auto Tarik Data BC 2-3 jika data diterima maka status akan berubah menjadi ( HOLD ) yang kemudian berubah menjadi ( STORED )
  2. Jika tidak memperoleh data maka Customer Service akan melakukan ON DEMAND, jika saat ,melakukan ON DEMAND data tidak tertarik juga maka Customer Service akan menginput secara manual
  3. Setelah itu Customer Service akan memeriksa Data Release P2 & PPC3 dan PPC2, apabila semuanya sudah RELEASE maka Petugas Manifest akan mengubah status di Sistem menjadi SET OUTGOING setelah itu input Nopol, Tanggal Keluar & Nomor dan Nama Pengurus serta Estimasi Tanggal Keluar
  4. Langkah selanjutnya adalah cek data apakah sudah released semua atau belum, jika belum maka koordinasi dengan P2 dan PPC3 untuk meminta Release
  5. Jika Release maka Petugas Operasional akan mencetak WO Release yang kemudian di berikan kepada Kerani Lapangan untuk melakukan perintah muat
  6. Selanjutnya Petugas Operasional melakukan Input Nopol , Tgl Keluar & Nomor dan Nama Pengurus, status akan berubah menjadi ( SET OUTGOING )g
  7. Setelah foto diambil dan di upload ke Sistem maka EMKL meminta Released ke BC
  8. Langkah selanjutnya adalah EMKL meminta Cap dan tanda tangan basah kepada petugas BC
  9. Setelah semua surat lengkap maka Supir akan Tap Kartu di AUTO GATE dan status berubah menjadi ( OUT )
Proses Set Outgoing
  • Memeriksa Data P2 dan PPC3 Release
    1. Petugas memeriksa data P2 dan PPC3 sudah Release, maka bisa langsung di SET OUTGOING
    2. Apabila PPC2 terkena HOLD maka petugas akan berkoordinasi dengan Petugas PPC2 untuk meminta Release
      • Buka Menu Outgoing di Menu PPC2
      • Kemudian Pilih salah satu Container yang akan Keluar Lalu tekan Released
    3. Set Outgoing
      • Buka Menu Stored
      • Kemudian Pilih salah satu Container yang akan Keluar Lalu Tap Gear dan Pilih Set Outgoing
      • Isi Nomor Polisi Truck dan Estimasi Tanggal Keluar Container kemudian pilih Save
  • Proses Outgoing
    1. Petugas memeriksa Verifikasi Dokumen BC 23, kemudian membawa Dokumen ke BC Gate untuk diperiksa
    2. Setelah mendapat tanda tangan BC pada Dokumen, maka petugas akan input Dokumen yang sudah benar tadi
    3. Setelah Informasi sudah di Input semua maka Petugas harus mencetak WO Relase yang kemudian di serahkan kepada Kerani Lapangan untuk melakukan Perintah Muat
    4. Petugas Operasional harus mengambil foto Segel Pelayaran, Segel BC, dan Foto Nomor Container
      • Buka Menu Outgoing
      • Kemudian Pilih salah satu Container yang akan Keluar Lalu Tap Gear dan Pilih foto yang akan diambil
    5. Setelah mendapatkan Release dari PPC2 maka Petugas Surat Jalan akan membuat Surat Jalan yang akan di serahkan ke EMKL untuk selanjutnya EMKL meminta Tanda Tangan dan Stempel Basah ke BC Gate
    6. Surat di serahkan kepada Supir, apabila Jadi keluar pada tanggal yang di input pertama maka Supir akan Tap Kartu dan Status akan berubah menjadi Out
    7. Jika Mobil tidak jadi keluar pada tanggal Estimasi maka Petugas akan melakukan SET OUTGOING lagi dan akan memasukan Estimasi tanggal baru
    8. Jika Kartu masih Valid maka Supir bisa Tap kartu dan Status akan berubah menjadi OUT, apabila tidak valid maka Petugas harus menyiapkan Kartu Baru
